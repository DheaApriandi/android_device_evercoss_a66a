	# CyanogenMod 13.0 for mt6582 (Evercoss A66A)

* Working
  * RIL 
  * Dual SIM 
  * Audio
  * Sensors
  * Wifi
  * MTP, ADB etc.
  * Bluetooth
  * Camera

* Not Working
  * GPS
  * FM-Radio
 
* Sync CyanogenMod source:

        # repo init -u git://github.com/dheaapriandi/android.git -b cm-13.0        
        # repo sync


* Compilation
       
        # source build/envsetup.sh    
        # brunch cm_a66a-userdebug
